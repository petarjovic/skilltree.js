Skill Tree Maker

Create simple yet robust skill trees.

Skill Tree Maker is an easy to use Javascript library designed to help you to create elegant and powerful "skill trees" in your web application.


Installation: Skill Tree Maker reqires no dependcies to work!
Simply download the library and include the skill-tree-maker.js script. Thats it!

Getting Started: Skill Tree Maker has two objects which are used in every skill tree. The SkillTree [ST] object and the SkillTreeNode [STN] object. STNs will always be instatiated as a part of an ST. A very basic skill tree requires the following object to be passed in. 

            { 
              svg: the blank svg which will be used to create the skill tree
              title: the title of the skill tree
              points: the number of points the user is given
              root: a STN object which is the root of the tree 
            }

It is recommended that the STN object be created right in the decleration to avoid surpurfluos code. A very basic skill tree node requires the following object to be passed in: 

            {
                title: the title of the STN
                children: a list of STN's direct descendants in the tree
            }

An example of a very simple three node tree is here: 


              const ThreeNodeTree = new SkillTree ({
                svg: document.getElementById("SkillTree"),
                title: "Three Nodes",
                points: 3,
                root: new SkillTreeNode ({
                  title: "Skill 1", 
                  children: [ 
                  new SkillTreeNode ({
                      title: "Skill 2"
                    }), new SkillTreeNode ({
                      title: "Skill 3"
                    })
                  ]
                })
              });

Finally, after creating the SkillTree as you like, you must run the .initialize() method!
Like so: 


              ThreeNodeTree.intitalize();

Further Documentation:
Due to the nature of constructing the skill tree modification after creaton (i.e. after running .initialize) is limited, but the creation process is very robust.

SkillTree Constructor: The SkillTree constructor has the following options:

svg
Type: HTML Element
Default: None
Description: Technically, the only *truly* required option. This takes an HTML SVG element that will be used to construct the skill tree.

title
Type: String
Default: ""
Description: The title of the skill tree which will be at the top, if left blank there will be no title.

points
Type: Number
Default: 0
Description: The number of points which can be spent on the skills within the skill tree.

root
Type: SkillTreeode
Default: None
Description: This takes the STN instance which will be the root node of the skill tree, if left empty there will be no tree.
Note: It is possible to create all STNs here to avoid surpurfluos code, see ThreeNodeTree example above.

respec
Type: boolean
Default: false
Description: If set to true it will be possible to click on an already unlocked skill to relock it, refunding all points spent on the skill.

free
Type: boolean
Default: false
Description: If set to true all skills within the skill tree will be free and points will not matter.

SkillTreeNode Constructor: The SkillTreeNode constructor has the following options:

title
Type: : String
Default: ""
Description: Title of the skill which will be displayed below the node.
Note: It is recommended that title of the skill not be longer than 10 characters as it will potentially overlap.

children
Type: : list[SKillTreeNode]
Default: : []
Description: The direct descendants of this node.
Note: It is recommended to not have more than 10 skills on the same level as the width of the tree will be extremely large and the UI may start getting wonky.
Note 2: SkillTreeNodes can be constructed direcrly in the decleration of the children. As seen in the ThreeNodeTree example.

cost
Type: Number
Default: 1
Description: The number of points which it will cost to unlock this skill.
Note: No effect if the SkillTree free option is set to true.

image
Type: String
Default: ""
Description: This takes a path to an SVG image which will be displayed on the node.
Note: It is recommended that the image be sqaure or almost square or there may be weird distortion.
