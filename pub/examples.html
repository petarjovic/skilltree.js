SkillTree.js

Home Documentation Examples

Create simple yet robust skill trees.

SkillTree.js is an easy to use Javascript library designed to help you to create elegant and powerful "skill trees" in your web application.


Installation: SkillTree.js reqires no dependcies to work!
Simply download the library and include the skilltree.js script. Thats it!
(Please do not actually use this library, it's bad!)

Getting Started: SkillTree.js has two objects which are used in every skill tree. The SkillTree object and the Perk object. Perks will always be instatiated as a part of a SkillTree. A very basic SkillTree requires the following object to be passed in. 

            { 
              canvas: the canvas element which will be used to create the skill tree
              title: the title of the skill tree
              points: the number of points the user is given
              root: a Perk object which is the root of the tree 
            }

A very basic Perk object requires the following object to be passed in: 

            {
                title: the title of the Perk
                children: a list of this Perk's direct descendants in the tree
            }

An example of a very simple three perk tree is here: 


              const ThreePerkTree = new SkillTree ({
                canvas: document.getElementById("SkillTree"),
                title: "Three Perk",
                points: 3,
                root: new Perk ({
                  title: "Skill 1", 
                  children: [ 
                  new Perk ({
                      title: "Skill 2"
                    }), new Perk ({
                      title: "Skill 3"
                    })
                  ]
                })
              });

Finally, after creating the SkillTree as you like, you must run the .initialize() method.
Like so: 


              ThreePerkTree.intitalize();

Further Documentation:

SkillTree Constructor: The SkillTree constructor has the following options:

canvas
Type: HTML Element
Default: None
Description: Technically, the only *truly* required option. This takes an HTML canvas element that will be used to construct the skill tree.

title
Type: String
Default: ""
Description: The title of the skill tree which will be at the top, if left blank there will be no title.

points
Type: Number
Default: 0
Description: The number of points which can be spent on the skills within the skill tree.

root
Type: SkillTreeode
Default: None
Description: This takes the Perk instance which will be the root perk of the skill tree, if left empty there will be no tree.
Note: It is possible to create all Perks here to avoid surpurfluos code, see the ThreePerkTree example above.

respec
Type: boolean
Default: false
Description: If set to true it will be possible to click on an already unlocked skill to relock it, refunding all points spent on the skill.

free
Type: boolean
Default: false
Description: If set to true all skills within the skill tree will be free and points will not be displayed.

Perk Constructor: The Perk constructor has the following options:

title
Type: : String
Default: ""
Description: Title of the skill; which will be displayed below the perk.

children
Type: : list[Perk]
Default: : []
Description: The direct descendants of this Perk.
Note 2: Perks can be constructed direcrtly in the decleration of the children. As seen in the ThreePerkTree example.

cost
Type: Number
Default: 1
Description: The number of points which it will cost to unlock this skill.
Note: No effect if the SkillTree free option is set to true.

image
Type: String
Default: ""
Description: This takes a path to an image which will be displayed on the perk.
Note: It is recommended that the image be sqaure or almost square or there may be weird distortion.

SkillTree Methods: The SkillTree class has the following methods:

.initialize(); : This method is to be called after all Perks have been instantiated and added to the skill tree.
Arguments: None

.addPoints(points: Number); : This method is to be called after all Perks have been instantiated and added to the skill tree.
Arguments: Takes a number which will be added to the current number of points, can be positive or negative.
